## Changes Made

### Left Sidebar (Charity Identity)
**Changed**: Location → Followers

**Before**:
- Updates
- Likes  
- Comments
- Location (Municipality)

**After**:
- Updates
- Likes
- Comments

**Before**:
1. Bin
2. Engagement Summary (This Month)
3. Latest Activity
4. Quick Actions
5. Community

**After**:
1. **Insights** (renamed from "This Month")
   - Total Likes
   - Comments
   - Posts
2. **Quick Actions**
   - New Update
   - View Campaigns
   - Dashboard
3. **Bin** (highlighted with gradient)
   - Description
   - View Bin button

**Removed**:
- Latest Activity section
- Community section
